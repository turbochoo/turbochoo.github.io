参考了archlinux 官网、各路大神等的配置，定制化了适合自己的开发环境。记录下,方便以后回顾。用了多年的macAIR发热量大的，让x240重新生, 安装完后居然温度稳定在49~54度，一点都不烫手，性能表现不错，很满意(thinkpad 跟linux 绝配)！

硬件配置如下：

  • thinkpadx240
  • 8G ram
  • 250G ssd

目标

  • 用于c++、java开发以及python等web开发
  • 搭配ssr酸酸乳进行科学上网
  • 实现休眠

当有时候需要提交某些文档或代码到git上，其他忽略不提交，这时候白名单机制就用得上了。比如，一个项目project，如下：

可以看到， git 只会添加 include_file文件与include_folder目录。

好了，我们先来了解 gitignore这个文件的语法,跟正则语法类似
，这边只说下该场景用到的标识：

  1. #开头为注释
  2. !开头标识非，gitignore文件本身也是非的意思，所以该行就是一条白名单规则
  3. * 标识为匹配目录下所有目录跟文件，单个标识为单层目录
  4. /为目录分割符
