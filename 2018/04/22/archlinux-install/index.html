how to install archlinux development enviroment - part1

参考了archlinux 官网、各路大神等的配置，定制化了适合自己的开发环境。记录下,方便以后回顾。用了多年的macAIR发热量大的，让x240重新生, 安装完后居然温度稳定在49~54度，一点都不烫手，性能表现不错，很满意(thinkpad 跟linux 绝配)！

硬件配置如下：

  • thinkpadx240
  • 8G ram
  • 250G ssd

目标

  • 用于c++、java开发以及python等web开发
  • 搭配ssr酸酸乳进行科学上网
  • 实现休眠
    -

基础安装

首先从官网下载 iso，校验sha1值，然后写入u盘(个人喜欢Easy2Boot 这个工具，直接copy进去即可)，设置u盘启动，进入命令行安装界面。接下来的这边列出关键步骤，具体看 安装指南

  1. 分区 选择需要安装的目标盘，划分如下：

    注意：

    • /boot 分区得标识为 boot flag , 后面会安装grub到该分区。
    • 除了wap分区得设置格式为83，其他默认84。

  2. 设置、挂载分区

  3. 安装base package，我用的是wifi所以，先通过wifi-menu命令启用网络,确保能访问到外网。

    • dialog iw wpa_supplicant 用于启动后方便设置网络
    • zsh 这个好用过bash
    • vim 方便命令行对文件进行编辑，习惯了vim（系统自带默认nano）
    • openssh 这个就不用说了，远程必备
    • git、make、gdb、autoconf、automake、patch 必备

    1. 上述准备好后，切换root & install grub2 & reboot

    2. 重启后，系统基本可用了，接下来就是安装软件与配置：
      设置root passwd,hostname,keymap、locale 中文环境设置参考、timezone、时间同步等。

    2. 通过wifi-menu 进行网络连接，再次更新软件包：

    • 先设置国内镜像，这样比较快 /etc/pacman.d/mirrorlist,启用几个国内的url，比如163等。
    • 设置 添加 /etc/pacman.conf
      打开 multilib 项
      添加archlinuxcn 源

    更新，并安装 archlinuxcn keyring

    到这一步系统基本已经安装好。接下来进入 图形界面篇
