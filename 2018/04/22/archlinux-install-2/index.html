安装图像界面的前，请参考
基本系统安装篇

linux 图形界面基本分为gtk跟qt，类似gnome，weightlight 的有lxde、xfce4可供选择。个人偏向tiling
window的manager,以前装过lua写的awesome,后来选择i3，个人感觉i3更好用。

一、图形涉及的软件包

本机使用的intel的芯片 选择 xf86-video-intel，nVidia安装xf86-video-nouveau即可

说明：

  1. xorg-xev 用于捕获按键的keycode, 方便用于设置key map
  2. xorg-xprop 用于捕获程序的class name,方便用于设置i3

二、字体、theme

说明：

  1. arc theme 个人感觉比较清爽
  2. nerd-fonts-complete、powerlevel9k 主要用于zsh
  3. 启动图形界面的时候加入中文支持，避免乱码(命令行界面支持中文输出安装fbterm)
  4. 推荐 wqy-microhei 作为系统默认字体，谁用谁知道
